# YA Books Explorer 📚

A Young Adult book discovery app — browse by genre, search by title or author, and view detailed book info.

🔗 **Live Demo:** https://ya-books-explorer.vercel.app

## Features

- Browse YA books by genre (Fantasy, Romance, Sci-Fi, Mystery, Horror, Contemporary)
- Search by title or author
- View book details — cover, description, rating, page count, and Google Books preview link
- Randomly featured books on the home page
- Responsive design for mobile and desktop

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS
- Google Books API
- Deployed to Vercel

## Run Locally

```bash
git clone https://github.com/Helenyixuanwang/ya-books-explorer.git
cd ya-books-explorer
npm install
```

Create a `.env` file in the root:
VITE_GOOGLE_BOOKS_API_KEY=your_api_key_here
Then:
```bash
npm run dev
```
